# React Todo List

![image](https://github.com/WMCharles/React-Todo-List/blob/master/Todo.png)

Clone the repository
 ```
 git clone git@github.com:WMCharles/React-Todo-List.git
 ```

Install packages 
```
npm install
```
## Available Scripts

In the project directory, you can run:

```
npm start
```

```
npm run server
```
